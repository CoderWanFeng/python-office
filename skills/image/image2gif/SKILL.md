# image2gif Skill

> 将图像转换为 GIF 格式

## 功能描述

将图像转换为 GIF 格式文件，负责处理图像数据并保存为 GIF。

## 所属分类

`office/skills/image/image2gif/`

## 调用方式

```python
from skills.image import image2gif

image2gif()
```

## 参数说明

无参数。

## 返回值

`None`

## 使用示例

```python
from skills.image import image2gif
image2gif()
```

## 原始函数

`office.api.image.image2gif`
